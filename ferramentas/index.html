Ferramentas

Aqui estão os links para algumas das ferramentas que vamos utilizar ao longo da disciplina.

Middleware de áudio para games

  • FMOD Studio: Ferramenta avançada para criação de áudio dinâmico para jogos.

Música

  • Bosca Ceoil: Ferramenta básica de criação musical para loops e arranjos. Exporta para MIDI. Tem uma versão online.
  • LMMS: Ferramenta mais avançada e gratuita para criação musical. Permite o uso de VSTs e controladores MIDI.

Editores de áudio

  • Audacity: Editor e gravador multitrack de áudio gratuito.

Geradores de SFX

  • BFXR: Gerador de efeitos sonoros retrô para jogos. Tem uma versão online e para download no mesmo site.
  • LabChirp: Gerador de SFX retrô.
